How to Calculate GPA

GPA is calculated using the formula:

GPA = Σ (Credit Hour × Grade Point) / Σ Total Credit Hours

Example: If you have two subjects:

  • Math: Credit Hour = 4, Grade Point = 3.6
  • Science: Credit Hour = 4, Grade Point = 3.2

Then GPA = (4×3.6 + 4×3.2) / (4+4) = (14.4 + 12.8)/8 = 27.2/8 = 3.4
